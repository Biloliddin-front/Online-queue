# Online Queue System

A Python-based online queue management system that allows users to join, manage, and track a virtual queue efficiently.

## Features

- Add users to queue
- Remove users from queue
- Display current queue
- Queue position tracking
- Simple command-line interface
- Data storage support

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- JSON/File Handling
